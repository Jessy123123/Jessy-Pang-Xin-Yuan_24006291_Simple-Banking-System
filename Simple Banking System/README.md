## Project Description
This project is a simple banking system developed using C# as a console application.  
It demonstrates basic Object-Oriented Programming (OOP) concepts such as classes, objects, constructors, encapsulation, and methods.

## Features
- Create a customer
- Deposit money into a bank account
- Withdraw money from the account
- Display customer information
- Display account information
- Display transaction details

## Classes Used
1. **Customer**
   - Stores customer ID, name, and phone number
   - Uses properties for encapsulation

2. **BankAccount**
   - Stores account number, holder name, account type, and balance
   - Displays account information

3. **Transaction**
   - Handles deposit and withdrawal operations
   - Calculates new account balance

4. **Program**
   - Main program that runs the banking system
